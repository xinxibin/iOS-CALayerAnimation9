# iOS-CALayerAnimation9


* 就是简单的两条线条实现
* 加入动画执行速度函数
* 动画开始结束回调 实现

* 使用方法

```objc
    let loadingView = XXBLoadingView(frame: CGRect(x: 0, y: 0, width: 50, height: 50))
    loadingView.center = self.view.center
    self.view.addSubview(loadingView)

```
